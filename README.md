## 💻 Sobre mí

Soy un desarrollador **full stack** con especialidad en **backend y bases de datos**. Aunque prefiero trabajar en el servidor, tengo una sólida experiencia en frontend que me permite entender y trabajar en ambos lados de la aplicación. Mi objetivo es crear soluciones completas, eficientes y bien integradas.

- 🐍 **Backend** he desarrollado en: Python, Java, Node.js
- 🗄️ Pasión por **bases de datos**: modelado avanzado, optimización, escalabilidad
- 🖥️ **Frontend competente**: React, Vue, JavaScript, HTML, CSS - entiendo la experiencia del usuario
- 🔗 Capacidad para **trabajar full stack**
- 💡 Compromiso con código limpio y arquitectura sólida
- 🚀 Siempre buscando nuevos retos y aprender tecnologías
